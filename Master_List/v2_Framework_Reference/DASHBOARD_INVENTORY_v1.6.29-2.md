## Dashboard Separation Rules

- Operational Dashboard (`Dashboard Quality Report`) is produced by quality, validation, timing, health, workflow, and signoff functions. It is an output and governance surface, not the configuration source of truth.
- Format Dashboard (`Format Dashboard`) is the configuration source of truth consumed by dashboard loaders, template builders, report formatters, validators, and workflow builders.
- Format Dashboard schemas, section titles, column order, sheet types, and header names are protected production architecture. Runtime values may be edited only where the dashboard loader and validators explicitly support edits.
- Operational Dashboard generated section titles and detail headers are protected. Detail rows are runtime output and should be regenerated by the owning workflow rather than manually maintained.
