# General Scripts

Scripts in this folder are non-production support utilities unless a file header or project README explicitly states otherwise.

They are not the governing source for the Master List production workflow and should not be used as production baselines for Master List releases.

General scripts still follow repository-wide Google Apps Script standards:

- preserve approved behavior;
- avoid unrelated changes;
- prefer batch reads/writes and in-memory processing;
- document any external dependencies, scopes, or manual setup requirements;
- do not treat utility scripts as public Master List APIs unless explicitly approved.
